# Portfolio

Welcome to my portfolio! This project showcases my skills and projects as a MERN stack developer with a focus on AI technologies. The design follows the principles of Neobrutalism, providing a unique and modern aesthetic.

## Technologies Used

- **Next.js**: A React framework for building server-side rendered applications.
- **TypeScript**: A superset of JavaScript that adds static types.
- **Shadcn**: A design system that enhances the UI components.

## Features

- Responsive design that works on various devices.
- Interactive components built with React.
- Showcase of my projects and skills.

## Getting Started

To get a local copy up and running, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   ```
2. Navigate to the project directory:
   ```bash
   cd your-repo-name
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:3000`.
